== NOVO METODO DE INICIALIZAÇÃO DE MATRIZ == 

. Esse novo método consiste em pegar de base do array básico lá, só que o array so tem o base e não possui outros arrays para interagir entre si, nesse caso basta seguir o mesmo principio: 

	int [] [] nome = new int {{1,2,3} , {1,2,3} , {1,3,4} }
